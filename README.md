# Gruvbuddy

A dark color theme for **Zed**, ported from the [gruvbuddy](https://github.com/tjdevries/colorbuddy.nvim) colorscheme by TJ DeVries.

Gruvbuddy is a gruvbox-inspired dark theme with a deep `#111111` background and a warm, muted palette.

## Installation

### Zed

Copy `themes/gruvbuddy.json` to your Zed themes directory:

```sh
# macOS / Linux
cp themes/gruvbuddy.json ~/.config/zed/themes/

# Then select Gruvbuddy from Zed's theme picker
```
